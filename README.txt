Pour lancer l'application, exécutez le fichier "QIz.py".

Pour creer son propre compte:
	appuyer sur "creer un compte	
	mettre son id et mdp	
	appuyer sur Fichier
	appuyer sur 'Accueil'
	se connecter avec son nouveau identifiant

Pour jouer à un quiz :
Appuyer sur le bouton correspondant à la (ou aux) réponse(s) que vous souhaitez sélectionner, puuis appuyez sur Valider.
Appuyer une deuxième fois sur un bouton réponse pour annuler la réponse.
Votre score sera plus élevé si vous répondez plus vite !
